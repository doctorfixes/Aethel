Privacy & Ethics

Data We Collect

The application stores a local student_profile.json file on your own machine. This file contains only:

  • Student level (integer)
  • Mastered topics (list of topic strings)
  • Known gap areas (topic and short note)

Every lesson interaction is also appended to a local audit.jsonl file for transparency and debugging. No personally identifiable information (PII) is collected or transmitted.

API Usage

Topic queries are sent to the Anthropic API using your private API key stored in a local .env file. This key is never committed to version control (see .gitignore). Anthropic's data usage and privacy policies apply to all prompts sent through their API.

Age Safety Protocol

All explanations are generated with age-appropriate filtering. The system prompt explicitly instructs the model to produce content suitable for the stated age (4–18). Requests outside this range are rejected by the server. Every lesson is additionally validated by a built-in QA tool before delivery.

Pedagogical Frameworks

Content is shaped by three evidence-based educational approaches:

  • Singapore CPA – concrete-pictorial-abstract progression
  • Estonian Logic – computational thinking and problem decomposition
  • US Recovery – scaffolded support and growth-mindset framing

Open Source

This project is open source. You can review all source code in the repository to verify data handling practices.
