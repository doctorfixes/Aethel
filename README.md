# 🌍 Universal AI Encyclopedia

An adaptive, agentic learning platform designed to bridge global education gaps.
It scrapes real-time data and reframes it using **Singaporean Math**, **Estonian Logic**,
and **US Recovery** pedagogical frameworks.

## 🚀 Quick Start

1. **Clone the repo:** `git clone <your-repo-url>`
2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   python -m playwright install chromium
   ```
3. **Set your API Key:** Create a `.env` file and add `OPENAI_API_KEY=your_key_here`.
4. **Launch:**
   ```bash
   python app.py
   ```
5. **Access:** Open http://localhost:8000 in your browser.

## 🧠 Features

- **Stateful Memory:** Tracks "Mastered Topics" and "Struggles" in a JSON-based profile.
- **Agentic Scraping:** Uses Crawl4AI to pull live data from Wikipedia.
- **Safety Protocol:** 2026-standard age-filtering and fact-checking.
