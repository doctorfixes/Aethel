# Æthel (Aethel)
**The Universal AI Encyclopedia & Adaptive Tutor**

Æthel is an agentic education platform designed to close global learning gaps. By synthesizing real-time data with the world's highest pedagogical standards—**Singaporean Math (CPA)**, **Estonian Systemic Logic**, and **Finnish Inquiry**—Æthel builds a persistent memory of a student's mastery to deliver a truly personalized "Recovery Curriculum."

### 🛡️ Core Philosophy
- **Provenance:** Every lesson is traced back to verified source material.
- **Statefulness:** A local-first memory layer that tracks mastery, not just clicks.
- **Sovereignty:** Students own their data; logic is processed transparently.

### 🛠️ Tech Stack
- **Engine:** Python / FastAPI
- **Intelligence:** LLM-agnostic (optimized for GPT-4o)
- **Scraper:** Crawl4AI (Chromium-based)
- **Memory:** Async JSON State Machine
